# Factorial-Excel-Custom-Function-AddIn

The **Factorial Excel Custom Function AddIn** is a tool that enhances your Excel experience by providing a custom function to calculate factorials. It allows you to compute factorials for all numbers from 1 to N, where N is a user-defined input. The results are populated in N consecutive rows within your Excel sheet.

## Features:

- **Factorial Calculation:** Quickly obtain factorials for a range of numbers from 1 to N.
- **Flexible Output:** Choose between displaying results in rows or cells based on your preference.
- **Cell Highlighting:** The ability to color the selected range in yellow for improved visibility.

## How to Use:

1. Install the Factorial AddIn in your Excel environment.
2. Use the custom function FACTORIALROW in your Excel formulas, specifying the desired input value N.
3. Explore the option to display results either in rows or cells.

## Run instructions:

```bash
    npm run build
```

```bash
    npm start
```
